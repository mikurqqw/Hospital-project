# Инструкция по запуску проекта "AI Health"

Для корректной работы приложения (включая локальные нейросети) на новом компьютере необходимо выполнить несколько шагов.

## Шаг 1: Установка необходимых программ
Убедитесь, что на компьютере установлены следующие программы:
1. **Python (версия 3.10 или выше)**. При установке ОБЯЗАТЕЛЬНО поставьте галочку `Add Python to PATH`.
2. **Node.js** (LTS версия).
3. **Ollama** (Движок для локальных нейросетей). Скачайте с официального сайта: https://ollama.com/download

## Шаг 2: Загрузка нейросетей
После установки Ollama откройте командную строку (cmd) и по очереди введите две команды, чтобы скачать модели:
1. `ollama pull qwen` (дождитесь окончания загрузки)
2. `ollama pull llava` (дождитесь окончания загрузки)

## Шаг 3: Запуск приложения
1. Откройте папку с проектом `healthcare-ai-app`.
2. Найдите и запустите файл **`start.bat`** (двойным кликом).
3. Скрипт автоматически:
   - Установит зависимости для фронтенда и соберет его.
   - Запустит нейросети в фоновом режиме.
   - Создаст виртуальное окружение Python, скачает библиотеки и запустит бэкенд.
4. Дождитесь надписи `ВСЕ СИСТЕМЫ РАБОТАЮТ!`. Приложение автоматически откроется по адресу: **http://127.0.0.1:8000**

## Возможные проблемы
* Если скрипт `start.bat` выдает ошибку при сборке фронтенда (конфликт версий `vite-plugin-pwa`), откройте командную строку в папке `frontend` и введите: 
  `npm install --legacy-peer-deps`
  После этого запустите `start.bat` заново.
